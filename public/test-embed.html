Embed Chatbot

This is a test page to verify the embed chatbot script is working correctly.

Instructions:
  1. Make sure your Next.js dev server is running (npm run dev)
  2. Open this page in your browser: http://localhost:3000/test-embed.html
  3. You should see a chat button in the bottom-right corner
  4. Click the button to open the chat widget
  5. Try sending a message to test the functionality

Note: Replace YOUR_USER_ID and YOUR_BASE_URL in the script tag below with your actual values from the Admin panel.
